Conway's Game of Life

WebAssembly Version with Bevy ECS 0.18.1

This is a WebAssembly version of Conway's Game of Life built with Bevy ECS and Rust, compiled to WASM using wasm-pack.

The canvas is created programmatically by the WASM module and will appear below.

Cellular Automaton Rules:

  • 🔴 Live cell with fewer than 2 neighbors dies (underpopulation)
  • 🟢 Live cell with 2 or 3 neighbors lives
  • 🔴 Live cell with more than 3 neighbors dies (overpopulation)
  • 🟢 Dead cell with exactly 3 neighbors becomes alive (reproduction)

The simulation runs automatically. Watch cells evolve based on Conway's rules!

Built with Rust, Bevy ECS 0.18.1, and WebAssembly | ESP32-HAL 1.1.0
